When starting a new session in this project, always check for a HANDOFF.md file at the project root.

If HANDOFF.md exists and its STATUS is `approved`:
1. Read the full HANDOFF.md
2. Execute the implementation plan step by step
3. Run `npm test` before committing any changes
4. Set STATUS to `review` when done
5. Report what was executed

If HANDOFF.md does not exist or STATUS is not `approved`, ask what task to work on.
